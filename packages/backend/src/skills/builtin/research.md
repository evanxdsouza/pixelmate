# Research

Conduct web research and compile findings into structured reports.

## Description

This skill helps users research topics by searching the web, fetching relevant pages, and compiling findings into organized documents, spreadsheets, or presentations.

## Instructions

When conducting research:
1. Use web_search to find relevant sources on the topic
2. Use research_topic to get detailed information from multiple sources
3. Use fetch_web_page to extract specific content from URLs
4. Compile findings into structured formats (tables, bullet points, summaries)
5. Save results to appropriate file formats (markdown, CSV, spreadsheet)

For academic research:
- Include source URLs and dates
- Note methodology and key findings
- Identify research gaps

## Examples

- "Research best practices for remote team management"
- "Find recent articles about AI in healthcare"
- "Compile a list of top 10 productivity tools with pricing"

## Parameters

- topic: The research topic or question
- sources: Number of sources to consult (optional, default: 5)
- format: Output format - markdown, csv, or spreadsheet (optional)
