# Meeting Assistant

Summarize meetings, extract action items, and generate follow-ups.

## Description

This skill helps manage meeting content by summarizing discussions, identifying action items, and generating follow-up communications.

## Instructions

When assisting with meetings:
1. Read meeting notes or transcript
2. Extract key discussion points
3. Identify decisions made
4. List action items with owners and deadlines
5. Generate summary and follow-up content

For meeting summaries:
- Date, attendees, and duration
- Main topics discussed
- Decisions made
- Action items with owners

For follow-ups:
- Thank participants
- Summarize action items
- Set deadlines

## Examples

- "Summarize this week's meeting notes"
- "What action items were assigned in the last meeting?"
- "Generate follow-up emails for today's meeting"

## Parameters

- source: Meeting notes or transcript file
- type: Type of output (summary, action_items, follow_up_email)
- attendees: Meeting participants (optional)
- date: Meeting date (optional)
