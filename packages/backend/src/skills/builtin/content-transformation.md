# Content Transformation

Convert content between different formats and platforms.

## Description

This skill transforms content between formats (video transcript to blog post, markdown to Word, CSV to JSON) and adapts it for different platforms.

## Instructions

When transforming content:
1. Read the source content
2. Understand the structure and key points
3. Transform to target format while preserving essential information
4. Optimize for the target platform (SEO for blogs, brevity for social media)

For transcript to blog posts:
- Add engaging introduction
- Use proper heading hierarchy
- Break into readable paragraphs
- Add SEO meta description

For content repurposing:
- Extract key points
- Adapt tone for new platform
- Maintain core message

## Examples

- "Convert this video transcript into a blog post"
- "Turn this report into a presentation"
- "Create social media posts from this article"

## Parameters

- source: Source file path or content
- sourceFormat: Original format (transcript, markdown, csv, json)
- targetFormat: Desired output format
- platform: Target platform if applicable (blog, social, email)
- options: Additional transformation options
