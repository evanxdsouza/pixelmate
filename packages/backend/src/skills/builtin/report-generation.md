# Report Generation

Create structured reports from various data sources.

## Description

This skill helps generate professional reports in multiple formats (Word documents, Markdown, HTML-ready content) from meeting notes, data files, and research.

## Instructions

When generating reports:
1. Gather relevant data from files or user input
2. Structure the report with clear sections: Executive Summary, Key Findings, Details, Recommendations
3. Use appropriate formatting: headings, tables, bullet points
4. Export to the requested format (Word document, markdown, CSV)

For business reports:
- Start with an executive summary
- Include data visualizations as tables
- End with actionable recommendations

For meeting reports:
- List attendees and date
- Summarize discussion points
- Document action items with owners

## Examples

- "Generate a weekly meeting summary report"
- "Create a quarterly business review document"
- "Make a report from these research notes"

## Parameters

- title: Report title
- sections: Report sections and content
- format: Output format (document, markdown, csv)
- audience: Target audience (optional)
