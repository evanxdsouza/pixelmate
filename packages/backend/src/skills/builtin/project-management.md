# Project Management

Organize tasks, track progress, and manage project documentation.

## Description

This skill helps manage projects by organizing tasks, creating project plans, tracking progress, and generating status reports.

## Instructions

When managing projects:
1. List all tasks and their status
2. Organize into phases or milestones
3. Identify dependencies and critical path
4. Track progress and blockers
5. Generate status reports

For task management:
- Create clear, actionable task descriptions
- Assign owners and deadlines
- Track status (todo, in_progress, done)

For project plans:
- Break into manageable phases
- Estimate effort and duration
- Identify resources needed

## Examples

- "Create a project plan for building a website"
- "What's the status of our current tasks?"
- "Generate a weekly project status report"

## Parameters

- project: Project name or description
- tasks: List of tasks (optional)
- output: Output format (spreadsheet, document, markdown)
- includeTimeline: Include timeline estimate (default: false)
