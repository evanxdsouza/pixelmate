# Data Analysis

Analyze data from files and generate insights.

## Description

This skill helps analyze data from various file formats (CSV, JSON, spreadsheets) and generate insights, summaries, and visualizations.

## Instructions

When analyzing data:
1. Read data from the appropriate source (CSV, JSON, spreadsheet)
2. Identify patterns, trends, and key metrics
3. Calculate aggregations (sums, averages, counts)
4. Generate summary tables and reports
5. Output results in appropriate format

For data analysis tasks:
- Start by understanding the data structure
- Clean and validate data as needed
- Use format_as_json or format_as_markdown to present results
- Create visualizations or summary tables

## Examples

- "Analyze this CSV file and find the top 5 values"
- "Calculate the average sales per month from the spreadsheet"
- "Generate a summary report of the data"

## Parameters

- source: Path to the data file
- analysis: Type of analysis to perform (summary, sort, filter, aggregate)
- output: Desired output format
