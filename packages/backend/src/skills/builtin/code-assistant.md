# Code Assistant

Help with code-related tasks, code review, and documentation.

## Description

This skill assists with programming tasks including code generation, review, explanation, and documentation.

## Instructions

When helping with code:
1. Read existing code files to understand context
2. Generate or modify code as requested
3. Explain code functionality clearly
4. Add comments and documentation
5. Follow best practices and coding standards

For code generation:
- Use appropriate language/framework conventions
- Include error handling
- Add comments for complex logic

For code review:
- Identify bugs and issues
- Suggest improvements
- Note security concerns

For documentation:
- Generate README files
- Add inline comments
- Create API documentation

## Examples

- "Write a function to parse this CSV file"
- "Review this code and suggest improvements"
- "Add documentation to this script"

## Parameters

- task: Task type (generate, review, explain, document)
- language: Programming language
- source: Source file path (optional)
- requirements: Specific requirements or constraints
