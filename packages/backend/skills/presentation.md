# Presentation

Create presentation slides and slideshows.

## Description

This skill helps create presentation content in various formats.

## Instructions

When creating presentations:
1. Structure content with clear slides
2. Use bullet points for key information
3. Include title slides
4. Keep each slide focused

## Examples

- "Create presentation slides for the quarterly review"
- "Make a 5-slide introduction to the project"
- "Generate slides with key points about the topic"

## Parameters

- filename: Name of the file (required)
- slides: Number of slides (optional, default: 5)
- topic: Main topic (required)
