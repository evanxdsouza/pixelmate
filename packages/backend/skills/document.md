# Document

Create and format text documents.

## Description

This skill helps create formatted text documents with headings, lists, and basic formatting.

## Instructions

When creating documents:
1. Use clear heading hierarchy
2. Organize content in sections
3. Include bullet points for lists
4. Keep paragraphs concise

## Examples

- "Write a meeting notes document"
- "Create a project proposal document"
- "Make a README file with installation instructions"

## Parameters

- filename: Name of the file to create (required)
- title: Document title (optional)
- content: Main content (required)
